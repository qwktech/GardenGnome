# GardenGnome Website

The front end to the GardenGnome webapp built in svelte.

## Dev Instructions

To start a dev server, run the below command and connect using the url `localhost:5173`
```bash
# download dependencies (only required on first run_
make web-init
# start the dev server
make web-dev
```

You can also open a terminal in a nodejs container by running the following
```bash
make web-terminal
```
